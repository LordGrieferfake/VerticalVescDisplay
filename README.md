
📟 VESC ESP32 Display
This project provides a display interface for VESC controllers using an ESP32.

💡 Credits
Inspired by [SimpleVescDisplayPublic](https://github.com/NetworkDir/SimpleVescDisplay?tab=readme-ov-file)

🛠 Installation & Setup
To get the project running, please follow these exact steps:
	1	📂 Replace Folder: Replace your old Arduino folder completely with the Arduino folder provided in this project.
	2	⚙️ Board Selection: In the Arduino IDE, go to Tools > Board and select ESP32 Dev Module.
	3	🚀 Flashing: Open the main file and flash it to your ESP32.
🔌 Hardware Connection
	•	The display connects to the VESC Controller via UART 🚥.

⚠️ Important Notes
	•	📏 Supported Hardware: Currently, only the 2.8-inch variant is supported. Support for other sizes is coming soon!
	•	🎨 Display Layout: Depending on your specific use case, you might need to manually adjust the positions of certain elements.
	•	📊 Parameters: Please note that the ranges of some parameters need to be adjusted manually to fit your requirements.
